Selamat datang di Halaman Utama

Ini adalah halaman utama website. Silakan navigasi ke halaman formulir atau tabel menggunakan menu di samping.


Website ini adalah situs web contoh yang dikembangkan sebagai proyek untuk tugas atau ujian dalam mata pelajaran pemrograman web. Website ini terdiri dari tiga halaman utama:

1. Halaman Utama (Homepage)

Halaman ini merupakan pintu masuk ke situs web. Ini berisi deskripsi singkat tentang website dan menu navigasi yang memungkinkan pengguna untuk beralih ke halaman lain, yaitu "Halaman Formulir" dan "Halaman Tabel."

2. Halaman Formulir (Form Page)

Halaman ini adalah halaman formulir pendaftaran. Pengguna dapat mengisi formulir dengan informasi seperti nama, NIM, program studi, email, password, jenis kelamin, tanggal lahir, dan alamat."

3. Halaman Tabel (Table Page)

Halaman ini menampilkan data mahasiswa dalam format tabel. Data mahasiswa yang ditampilkan termasuk nama, NIM, program studi, email, password, jenis kelamin, tanggal lahir, dan alamat. Data ini akan ditambahkan secara dinamis saat pengguna mengisi formulir di "Halaman Formulir."
